# Tech Stack and Coding Conventions

## Tech Stack
- Language: JavaScript/TypeScript
- Framework: AstroJS
- Package Manager: bun
- Testing: Vitest (recommended for Astro)

## Coding Conventions
- Use TypeScript for type safety
- Follow Astro's component-based architecture (.astro files)
- Use CSS Modules or Tailwind CSS for styling
- Maintain responsive design principles
- Implement accessibility best practices (ARIA labels, semantic HTML)
- Use functional components for reusable UI elements
- Follow Prettier formatting with 2-space indentation
- Keep components small and focused (single responsibility)
- Use descriptive file and variable names
- Implement proper error handling and loading states
- Follow SEO best practices for landing pages
